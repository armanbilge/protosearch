Demo: Repo Search

This search indexes metadata on GitHub repos from typelevel, http4s, circe, davenverse, and typelevel affiliate projects. Each repo has the following fields indexed: name, fullName, description, topics. To query a field, prefix your query with the field name, e.g. name:cats The default field is the description.

Try some of these examples:

cats fs2 json

(csv json) AND streaming

parsing AND NOT (json OR jawn)

fullName:[c TO e]

topics:(fp functional-programming)
